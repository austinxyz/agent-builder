# 皮克球段位标准

**来源：** USA Pickleball 官方段位说明 / Pickleball Canada 评级表 / DUPR 系统
**适用场景：** Step 1（评估水平）/ Step 2（制定目标）
**最后更新：** 2026-06-19

---

## 核心内容

### 段位总览

| 段位 | 标签 | 核心特征 |
|------|------|---------|
| 2.0 | 绝对新手 | 接触皮克球不久，球不能稳定落点，基本规则还在学习 |
| 2.5 | 初级 | 能打完整场球，但失误频繁，没有战术意识 |
| 3.0 | 中级入门 | 发球和回球稳定，开始意识到战术，第三拍drop不稳定 |
| 3.5 | 中级 | 能打出稳定的第三拍drop，有意识的落点选择，dink开始稳定 |
| 4.0 | 中高级 | 软球/硬球全面，有全场意识，能在压力下保持稳定 |
| 4.5+ | 高级/职业 | 超出本 agent 服务范围 |

---

### 各段位详细特征

#### 2.0 · 绝对新手
- 球落点不稳定，打哪里不能控制
- 基本规则（厨房规则/发球规则）还在记忆中
- 动作靠直觉，没有标准姿势
- **升段关键：** 稳定发球 + 学会移动到厨房线

#### 2.5 · 初级
- 能完成一场完整比赛
- 发球基本稳定，但回球失误多
- 开始学习volley，dink偶尔能打
- 没有第三拍意识
- **升段关键：** 减少非受迫性失误，建立回球稳定性

#### 3.0 · 中级入门
- 发球和基础回球稳定
- 开始理解第三拍概念，但执行不稳定
- dink能打，但不稳定，落点控制弱
- 战术意识萌芽，但执行跟不上
- **升段关键：** 第三拍drop成功率达到50%+，dink稳定10拍以上

#### 3.5 · 中级
- **第三拍drop是这个段位的标志技术**——能从底线稳定送球进厨房
- dink稳定，开始有落点意识（打对手反手/空档）
- 移动步法基本到位，能守住厨房线
- 开始打战术球，不只是稳球
- **升段关键：** 第三拍drop成功率70%+，能主动创造得分机会

#### 4.0 · 中高级
- 软球（dink/drop）和硬球（drive/smash）都能运用
- 全场意识：知道自己在场上哪里，对手弱点在哪
- 能在比赛压力下保持技术稳定
- 开始使用旋转球和战术变化
- **升段关键：** 进入专业训练体系，超出本 agent 常规服务范围

### 段位判断快速参考

**agent 在 Step 1 判断段位时参考：**

| 用户描述 | 初判段位 |
|---------|---------|
| "打了不到3个月，经常失误" | 2.0-2.5 |
| "打了半年以上，能打完整场，但进步不明显" | 2.5-3.0 |
| "第三拍老是出界/下网" | 3.0（卡在升3.5的节点） |
| "dink打得还行，但对手强一点就垮" | 3.5 |
| "比赛稳定，但遇到4.0+就很吃力" | 3.5-4.0 |

---

## 在对话中如何引用

> agent 引用示例：「根据你描述的情况——打了一年，第三拍经常出界——你目前大约在 3.0 段位。3.0 升 3.5 的核心就是第三拍 drop，这是我们训练计划的重点。」
