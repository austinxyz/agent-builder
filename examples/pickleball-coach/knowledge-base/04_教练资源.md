## 核心内容

### 为什么要找认证教练

皮克球起步晚，大量"教练"只是打得还不错的玩家，没有系统教学训练。找认证教练能确保：
- 教学方法经过专业培训，不是靠感觉教
- 有责任心和职业标准
- 知道如何针对不同段位设计训练

---

### 主要认证机构

| 机构 | 全称 | 特点 | 查找教练入口 |
|------|------|------|------------|
| **PPR** | Professional Pickleball Registry | USA Pickleball 官方合作，最大认证机构，10,000+认证教练，覆盖14国 | [pprpickleball.org](https://pprpickleball.org) |
| **PCI** | Pickleball Coaching International | Mark Renneson 创立，教育哲学导向，注重结构化教学方法 | [pickleballcoachinginternational.com/pages/member-directory](https://www.pickleballcoachinginternational.com/pages/member-directory) |
| **IPTPA** | International Pickleball Teaching Professional Association | 评估严格，较小众 | iptpa.com |

---

### 如何筛选好教练（筛选标准）

**基本门槛：**
- [ ] 有 PPR / PCI / IPTPA 其中一个认证
- [ ] 能描述教学流程（不只是"我带你打"）
- [ ] 第一节课会评估你的段位和短板

**加分项：**
- [ ] 能根据你的短板定制训练计划（不是给所有人同样内容）
- [ ] 提供视频回放或技术分析
- [ ] 有明确的升段目标和时间线

**红旗（不推荐）：**
- ❌ "我打到X.X分，肯定能教你" → 打得好≠教得好
- ❌ 没有任何认证，只说"打了N年"
- ❌ 第一节课就只是陪打，没有任何技术反馈

---

### 查找教练的方式

1. **PPR 官网** → Find an Instructor → 输入邮编
2. **PCI 成员目录** → [pickleballcoachinginternational.com/pages/member-directory](https://www.pickleballcoachinginternational.com/pages/member-directory)
3. **PlayPickleball.com** → Find Instructor → [playpickleball.com/find-instructors-by-name](https://www.playpickleball.com/find-instructors-by-name/)
4. **当地俱乐部/球场** → 询问是否有认证教练驻场
5. **Pickleheads App** → 部分教练在平台注册

### 费用参考（美国，2026）

| 类型 | 价格区间 |
|------|---------|
| 私教（1对1） | $60-120/小时 |
| 半私教（2-3人） | $35-60/人/小时 |
| 团课（4+人） | $15-30/人/次 |
| 线上课（视频）| $20-50/课时 |

---

### 没有本地认证教练怎么办

1. **PCI/PPR 线上课程** → 有视频教学，可自学结构化内容
2. **Selkirk TV**（免费）→ Mark Renneson 的系列教学视频
3. **The Dink / Pickleball Kitchen** → 高质量免费教学内容
4. **本地俱乐部高手带练** → 不是最优但比随机打球强，提前说明"我想学，希望你能给反馈"

## 在对话中如何引用

> agent 引用示例：「找教练首先查 PPR 官网，输入你的邮编找附近认证教练。第一节课问他：'你会先评估我的段位吗？你会给我定制化的训练计划吗？'——如果回答含糊，换一个。」
