## 核心内容

### 赛事类型（从入门到进阶）

| 类型 | 难度 | 适合段位 | 特点 |
|------|------|---------|------|
| 俱乐部内部赛 | ⭐ | 2.5-3.0 | 最友善，认识的人，低压力，好起步 |
| 本地休闲赛 / Fun Tournament | ⭐⭐ | 2.5-3.5 | 社区级别，通常有分段报名 |
| USA Pickleball 认证赛事 | ⭐⭐⭐ | 3.0+ | 正式计分，进入 DUPR 系统，有排名 |
| Golden Ticket Tournament | ⭐⭐⭐⭐ | 3.5+ | 通往全国赛资格赛 |
| USA Pickleball 全国赛 | ⭐⭐⭐⭐⭐ | 4.0+ | 顶级业余赛事，2026年10月31日-11月8日，圣地亚哥 |

### 如何查找本地赛事

1. **USA Pickleball 官网** → [usapickleball.org/tournaments](https://usapickleball.org/tournaments/) → 按州/地区筛选
2. **PickleballMAX 赛事日历** → [pickleballmax.com/2025/12/pickleball-tournament-calendar-2026](https://www.pickleballmax.com/2025/12/pickleball-tournament-calendar-2026/) → 2026全年日历
3. **Pickleheads App** → Events 板块 → 本地赛事
4. **本地俱乐部布告栏 / 群** → 最小型赛事只在这里发布

---

### 报名流程（USA Pickleball 认证赛事）

1. 确认自己的 DUPR 评级（无评级可以用自评段位报名）
2. 前往 [usapickleball.org/tournaments](https://usapickleball.org/tournaments/) 找目标赛事
3. 点击赛事 → 查看报名入口（通常是 PickleballBrackets 或 CaptainU 平台）
4. 选择组别（按段位+年龄+单/双打）
5. 付报名费（本地赛 $20-50，大型赛事 $50-150）
6. 确认搭档（双打）

### 参赛前准备（赛前X周训练重点）

| 距比赛时间 | 训练重点 |
|----------|---------|
| 4-6周前 | 专攻最大短板，集中突破一个技术 |
| 2-3周前 | 模拟比赛对打，在压力下打平时练的技术 |
| 1周前 | 减少高强度训练，保持手感，重点打drill |
| 比赛当天 | 热身15-20分钟（drill），不要在热身时打全力 |

---

### 比赛规则要点（业余选手常犯错）

| 规则 | 说明 |
|------|------|
| 厨房规则 | 截击（空中接球）不能踩厨房线或进厨房区——脚踩线也算违规 |
| 发球规则 | 发球必须下手、球拍接触点低于腰部、球拍头低于手腕 |
| 双跳规则 | 发球方和接发球方各自必须先让球弹一次才能截击（Two-Bounce Rule） |
| 分数报告 | 每次发球前必须报分（发球方分数-接球方分数-发球次序） |
| 换边 | 单打每11分换边；双打看具体赛事规则 |

### DUPR 评级系统

- **DUPR**（Dynamic Universal Pickleball Rating）= USA Pickleball 官方评级
- 参加认证赛事后自动更新
- 没打过比赛可以通过 app 自评，作为起始评分
- 评分从 2.0 到 8.0，精确到小数点后两位
- App：[DUPR 官网](https://mydupr.com/)

---

### 赛后复盘框架

比赛结束后，agent 用这个框架引导复盘：

1. **得分回合** → 你赢得的分大多是靠什么打出来的？
2. **失分回合** → 你丢的分主要是哪种失误（主动失误/被动失误/对手打赢？）
3. **第三拍执行率** → 这场第三拍有几成落进厨房？
4. **心理状态** → 有没有因为失分连续丢局？怎么调整的？
5. **下阶段重点** → 根据以上，训练计划需要调整哪里？
